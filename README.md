# ts-node-skeleton

Boilerplate for ts-node.

## What's included

- ts-node
- TypeScript
- Prettier
- Recommended VSCode configuration

## Usage

```bash
cd app_name
wget -O - https://github.com/mizdra/ts-node-skeleton/archive/master.tar.gz | tar xzvf - --strip=1

## Init project
yarn install
yarn dev
```
